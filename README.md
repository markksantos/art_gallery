# Terminal Art Gallery

An interactive terminal art gallery featuring 6 generative art animations — all in pure Python using `curses`.

## Animations

1. **Matrix Rain** — Cascading green characters inspired by The Matrix
2. **Starfield** — A 3D starfield flying through space
3. **Fireworks** — Colorful firework rockets and particle bursts with gravity
4. **Game of Life** — Conway's Game of Life with color-coded cell age
5. **Plasma Waves** — Smooth, undulating plasma effect with rainbow gradients
6. **Maze Generator** — Watch a maze carve itself in real-time using recursive backtracking

## Requirements

- Python 3.6+
- A terminal with color support (256-color recommended)

No external dependencies — only Python standard library modules are used.

## Usage

```bash
python3 art_gallery.py
```

## Controls

| Key | Action |
|---|---|
| `Left` / `Right` | Switch between animations |
| `1`–`6` | Jump to a specific animation |
| `Space` | Pause / Resume |
| `r` | Reset current animation |
| `q` | Quit |
